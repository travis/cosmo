
Cosmo Sharing Server 0.2.5
Release Notes
        

UPGRADING FROM A PREVIOUS RELEASE
=================================

If upgrading from 0.2.4, no configuration changes are
required. Repositories created by 0.2.4 may be used by this version
with no changes.

Releases prior to 0.2.4 required Java 1.4, but starting with 0.2.4,
Cosmo now requires Java 5.0 (1.5).


NEW FEATURES
============

4345: Responses to CMP user create and update operations now contain
      ETags
4817: CMP user create and update operations return specific status
      codes for "username in use" and "email in use" errors
4830: new CMP operations for authenticated users to view and update
      their account information
4854: User first name, last name and email may now be up to 128 bytes
      long.
4864: Avoid NPE when PROPPATCHing a dav displayname with no value


BUGS FIXED
==========

4858: Memory leak caused by not logging out of repository sessions is
      plugged
4794: Tomcat no longer creates unwanted log files
